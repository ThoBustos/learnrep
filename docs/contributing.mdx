LearnRep is MIT licensed. The repo is at [github.com/ThoBustos/learnrep](https://github.com/ThoBustos/learnrep).

## Structure

```
apps/
  cli/       CLI (Node.js, Commander)
  web/       Next.js app
packages/
  core/      Generator, evaluator, types (shared)
```

## Running locally

```bash
pnpm install
pnpm dev
```

## Issues and PRs

Open issues or PRs on GitHub. Bug reports, feature requests, and doc improvements are all welcome.
