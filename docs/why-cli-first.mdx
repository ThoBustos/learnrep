---
title: Why CLI-first
description: Why LearnRep starts in the terminal.
---

Most learning tools ask you to leave your flow. LearnRep stays in it.

Developers already live in the terminal and their IDE. When you finish a coding session or read a doc, the content is right there. `lr generate` turns it into a quiz without switching context.

The CLI also makes LearnRep composable. Pipe in any text, script it into a workflow, or let an LLM agent call it automatically after a session ends.

The web app and MCP server are addons — the CLI is the core.
