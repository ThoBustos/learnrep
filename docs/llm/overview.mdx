---
title: Using with LLM apps
description: How to wire LearnRep into Claude Code, Cursor, or any LLM harness.
---

LearnRep CLI is designed to be called by LLM agents. After a coding session, a review conversation, or any learning moment, an agent can pipe the content straight into `lr generate` and return a quiz link.

The MCP server (`@learnrep/mcp`) exposes the same functionality as a tool call — no shell required.

See [Copy-paste config](/llm/config) for ready-to-use system prompt snippets.
